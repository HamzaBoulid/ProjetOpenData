Lancement application WEB 

-------------------------

1 ) 

Télécharger composer

https://getcomposer.org/download/

2 ) 

Avoir installé symfony

https://symfony.com/download

(Php est requis https://www.php.net/downloads.php)

3 ) 

Utiliser VS Code ou un autre terminal php

se rendre à la racine du document

puis lancer : 

composer update (facultatif)

composer install

php bin/console server:run

4 ) 

se rendre sur 127.0.0.1:8000 sur votre navigateur
